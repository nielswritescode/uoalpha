Events are often associated with a cost type like 'free', 'donation based' or 'NOTAFLOF' (no one turned away for lack of funds). Events following UOPs will need new cost types. 

Let's define 'Unconditionally Given' for now. Important to note is that we just define it for this philosophy and that it is subject to change as the philosophy grows. We could also define it as 'UG in the spirit of UOalpha' or 'UG(UOalpha)'. This will become relevant when more UOPs have emerged and there is a need for differentiate between different types of UG.

An event is UG when the following conditions are met:

- There is no expectation of reciprocity.
- The event is not used for marketing.
- The event is not used for political or ideological reasons.
- The event is not used to change people's spiritual view. Obviously the event itself proposes spiritual change. There is a fine line between offering a new view and expecting people to accept your view. 
- The event strives to have as little conditions as possible.
